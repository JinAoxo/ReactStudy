입력폼 템플릿 3종

  • 템플릿 3종은 마크업 구조는 완전히 동일, 컨테이너 클래스만 교체 사용
    유형별 Desktop 버전 디자인만 다르며, 모바일은 열드롭형으로 3종 모두 동일
  • [웹접근성] label for - id 매칭 및 caption 올바른 내용 입력 필수
  • [웹접근성] label 매칭시 시각적으로 확인 가능한 텍스트가 화면상에 존재하는 경우 해당 텍스트에 직접 label을 매칭,
    매칭되는 텍스트가 없는 경우에는 input title="" 로 대체텍스트 처리
  • label에 .sr-only 클래스는 디자인 체크박스 이외 사용 금지. 보이지 않는 대체텍스트는 title="" 로 처리
